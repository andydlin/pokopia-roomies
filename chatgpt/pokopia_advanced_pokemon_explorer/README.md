# Pokopia Advanced Pokémon Explorer Pack

This zip is designed to hand directly to Codex so it can scaffold a polished "Advanced Pokémon Explorer" web app.

## What is inside

- `CODEX_PROMPT.md` — a strong one-shot prompt for Codex
- `PRODUCT_BRIEF.md` — product goals, user problems, scope
- `FEATURE_REQUIREMENTS.md` — detailed feature requirements
- `DATA_MODEL.md` — suggested schema and field definitions
- `UI_NOTES.md` — page structure, components, UX details
- `SAMPLE_DATA_SCHEMA.json` — starter JSON schema and sample records
- `SUCCESS_CRITERIA.md` — what “done” looks like

## Recommended use with Codex

1. Upload the whole zip to Codex.
2. Start with `CODEX_PROMPT.md`.
3. Tell Codex to use the other files as the project brief.
4. Ask it to scaffold the app first, then iterate.

## Suggested stack

- Next.js
- TypeScript
- Tailwind CSS
- Local JSON dataset first, easy to replace later
