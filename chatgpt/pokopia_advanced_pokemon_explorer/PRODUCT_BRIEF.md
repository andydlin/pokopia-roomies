# Product Brief

## Product
Pokopia Advanced Pokémon Explorer

## Problem
Static guides are useful for raw lookup, but they are slow for comparison. Players often need to cross-reference ideal habitats, specialties, favorites, and item preferences across many Pokémon.

## Opportunity
Build a dedicated explorer that turns raw reference data into a faster decision-making tool.

## Primary user jobs
- Find Pokémon by ideal habitat
- Find Pokémon by favorite category
- Find Pokémon with a specific specialty
- Compare multiple Pokémon quickly
- Discover related Pokémon without manual page hopping

## Target user outcomes
- Faster browsing
- Better planning
- Fewer tabs open
- Easier discovery of good combinations

## In-scope
- Searchable explorer
- Rich filtering
- Sorting
- Detail view
- Related Pokémon suggestions
- Sample flexible data model

## Out-of-scope for v1
- Save groups
- Full compatibility scoring engine
- User accounts
- Real scraping pipeline
- Live syncing from external sites

## Product principles
- Fast to scan
- Easy to compare
- Built for expansion
- Better than a wiki for exploration
